## Surcharge Table Schema
Each record in `surcharge_table` has these fields:
{
  "sku": "RB-COLOR-ALL-BLK",
  "display_name": "All Black",
  "color": "Black",
  "applies_to": "Job | Frame | Frame Only | Louvers | Premium Solid Roof Panels",
  "pricing_unit": "per_bay | percent | per_sq_ft | Unit",
  "value": 800,
  "notes": "string"
}

Match rules:
- Match by product_id (already filtered) + color (case-insensitive on color).
- Use `applies_to` to determine which component the surcharge covers.
- If no match exists for a selected finish:
  - Return status="needs_info"
  - questions MUST include exactly:
    ["Missing finish surcharge SKU/price for the selected finish. Please update surcharge_table."]
  - pricing outputs must be null/empty.

## Quantity + Math Rules (STRICT)
Read bay data from `input_from_s03`:
- In single mode: use `result_single.total_bays`, `result_single.bay_width_ft`, `result_single.bay_length_ft`
- In comparison mode: use `result_keep` and `result_swap` separately

For each matched surcharge line:
- If pricing_unit="per_bay":
  qty = total_bays
  ext_price = value * qty
- If pricing_unit="percent":
  qty = 1
  ext_price = round(base_system_total * (value / 100), 2)
  (Use base_system_total_keep / base_system_total_swap in comparison mode)
- If pricing_unit="per_sq_ft":
  total_sqft = bay_width_ft * bay_length_ft * total_bays
  qty = total_sqft
  ext_price = value * qty
- If pricing_unit="Unit":
  qty = 1
  ext_price = value * qty

Totals:
- finish_total = sum(ext_price) for all matched surcharge lines

Comparison mode:
- If comparison_mode=false: compute single only (priced_keep and priced_swap = null).
- If comparison_mode=true: compute keep and swap independently (priced_single = null). Do NOT combine.
